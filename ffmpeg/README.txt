Place ffmpeg.exe here for a full-repo checkout (default from download script).

Portable GUI build: use the same script with a target folder, e.g.

  powershell -ExecutionPolicy Bypass -File .\scripts\download_ffmpeg.ps1 -DestDir ".\gui\ffmpeg"

Then gui\ffmpeg\ffmpeg.exe sits next to gui\iptv-gui.exe.
